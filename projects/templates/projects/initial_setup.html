First User Becomes Admin

The first account to connect via GitHub will automatically become the server administrator with full access to all features and settings.

What you can do after connecting:

  • Browse and connect your GitHub repositories
  • Trigger Docker image builds from any commit
  • Monitor build status and view logs
  • Access the admin interface
  • Use the REST API for automation
