GitHub OAuth Not Configured

GitHub authentication is not set up yet. An administrator needs to configure the GitHub OAuth application before users can connect.

For Administrators

To configure GitHub OAuth, follow these steps:
  1. Create a GitHub OAuth App at github.com/settings/developers
  2. Set the callback URL to: http://localhost:8000/accounts/github/login/callback/
  3. Run the setup command: 
    export GITHUB_CLIENT_ID="your_client_id"
export GITHUB_CLIENT_SECRET="your_client_secret"
python manage.py setup_github_oauth

See GITHUB_OAUTH.md for detailed setup instructions.
